ROI Coverage

Usage

roi_coverage(rois, fixations)

Arguments

rois
an object of class gridded_rois
fixations
coordinate vectors of fixations. This can be specified as a 2-column matrix x, a list x with two components

Description

Calculates the proportion of ROI cells covered by fixations
